# -
Проект на 2 месяца
Бот для поиска и загрузки музыки
Есть функция найти музыку куда после нажатия вас просят скинуть отрывок песни не менее 5 символов
после отправки через lyrics он ищет 5 самых похожих песен по тексту и предоставляет вам список 
следующая функция загрузки музыки, она работает через ffmpeg + youtube поэтому для работы в рф нужен обход или впн
после того как вы отправляете название песни, он находит 5 лучших результатов и предоставляет вам с ссылками на сами песни
после вам предлогают ввести от 1 до 5 для выбора подходящего трека, затем бот скачивает видео, форматирует его в аудио и скидывает музыку
(До замедления телеграмма все работало гораздо лучше и быстрее)
