Music Search Bot

Telegram бот для поиска песен по отрывку текста через Genius API.

Что умеет бот

Находит песню по нескольким строкам текста. Показывает название, исполнителя и ссылку на полный текст. Работает через простые кнопки.

Как работает

Пользователь нажимает кнопку "Найти музыку по тексту". Бот просит отправить отрывок песни. Пользователь пишет несколько строк. Бот ищет через Genius API и возвращает список найденных песен. Каждая песня сопровождается ссылкой.

Если песня не найдена, бот предлагает попробовать другой отрывок или использовать более уникальные слова.

Команды

/start - показать главное меню с кнопками
/help - справка по использованию
/cancel - отменить текущее действие

Технологии

Python 3.8+, библиотеки python-telegram-bot (работа с Telegram), lyricsgenius (поиск песен), python-dotenv (хранение токенов).

Установка

Скопируйте репозиторий:
git clone https://github.com/username/music-search-bot.git
cd music-search-bot

Установите зависимости:
pip install -r requirements.txt

Создайте файл .env и добавьте токены:
TELEGRAM_BOT_TOKEN=ваш_токен
GENIUS_ACCESS_TOKEN=ваш_токен_genius

Запустите бота:
python bot.py

Где взять токены

Токен Telegram бота - у BotFather в Telegram. Токен Genius API - на сайте genius.com/api-clients.

Структура проекта

bot.py - основной код бота
requirements.txt - список библиотек
.env - токены (не загружать в репозиторий)
